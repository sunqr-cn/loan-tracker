# 还款提醒服务 - 产品设计与落地计划

## 一、项目概述

### 1.1 背景
现有公积金贷款还款计划管理应用已实现：
- 前端：React + TypeScript + Zustand
- 后端：Cloudflare Worker + D1 数据库
- 数据接口：`https://loan-tracker-api.pages.dev/api/data`

用户希望增加**还款提醒功能**，在还款日前自动发送邮件提醒。

### 1.2 目标
- 搭建独立的还款提醒服务
- 不影响现有项目代码逻辑
- 数据与现有系统互通（读取同一份贷款数据）
- 完全免费（GitHub Actions + Resend 邮件服务）

### 1.3 核心约束
- **不修改现有前端代码**
- **不修改现有 Worker 代码**
- 提醒服务独立部署，仅读取现有 API 数据

## 二、产品设计

### 2.1 功能范围

#### 2.1.1 核心功能
1. **还款提醒邮件**
   - 还款日前 3 天发送提醒邮件
   - 邮件内容包含：本期还款金额、还款日期、剩余本金
   - 支持多用户（通过邮箱配置）

2. **用户配置管理**
   - 通过配置文件管理提醒邮箱
   - 支持多个邮箱地址
   - 可配置提醒天数（默认 3 天）

#### 2.1.2 非功能需求
- 每月运行成本：0 元（GitHub Actions 免费额度 + Resend 免费额度）
- 可靠性：失败时记录日志，不阻塞下次执行
- 可观测性：每次执行记录结果（成功/失败/跳过）

### 2.2 用户场景

**场景 1：用户配置提醒**
1. 用户在 GitHub 仓库的 `reminder-config.json` 中填写邮箱
2. 提交代码后，GitHub Actions 每天自动检查
3. 还款日前 3 天收到提醒邮件

**场景 2：系统自动运行**
1. GitHub Actions 每天 UTC 00:00 触发
2. 调用现有 API 读取贷款数据
3. 计算下次还款日期
4. 如果距离还款日 <= 3 天，发送邮件
5. 记录执行日志

### 2.3 数据流

```
┌─────────────────┐
│  现有 Worker API │  (loan-tracker-api.pages.dev)
│  /api/data      │
└────────┬────────┘
         │ 读取
         ▼
┌─────────────────┐
│  提醒服务脚本    │  (GitHub Actions)
│  check-reminder.js
└────────┬────────┘
         │ 判断是否需要提醒
         ▼
┌─────────────────┐
│  Resend API     │  (发送邮件)
│  免费额度 3000/月│

```

### 3.3 配置文件设计

**reminder/config.json**
```json
{
  "emails": ["user@example.com"],
  "reminderDays": 3,
  "apiUrl": "https://loan-tracker-api.pages.dev"
}
```

### 3.4 脚本逻辑

**check-reminder.js**
```javascript
// 1. 读取配置
const config = require('./config.json');

// 2. 调用 API 获取数据
const response = await fetch(`${config.apiUrl}/api/data`);
const data = await response.json();

// 3. 解析还款计划
const schedule = data.data.schedule;
const today = new Date();
const nextPayment = schedule.find(s => !s.paid && new Date(s.date) > today);

// 4. 计算距离还款日天数
const daysUntilPayment = Math.ceil((nextPayment.date - today) / (1000 * 60 * 60 * 24));

// 5. 判断是否需要提醒
if (daysUntilPayment <= config.reminderDays) {
  // 发送邮件
  await sendReminderEmail(config.emails, nextPayment, daysUntilPayment);
}
```

### 3.5 GitHub Actions 工作流

**.github/workflows/reminder.yml**
```yaml
name: 还款提醒

on:
  schedule:
    - cron: '0 0 * * *'  # 每天 UTC 00:00
  workflow_dispatch:      # 支持手动触发

jobs:
  check-reminder:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '20'
      
      - name: Install dependencies
        run: cd reminder && npm install
      
      - name: Check reminder
        env:
          RESEND_API_KEY: ${{ secrets.RESEND_API_KEY }}
        run: cd reminder && node check-reminder.js
```
