# 还款账户功能 - 实施计划

## Context

用户需要为贷款还款追踪应用新增"还款账户"功能，模拟银行卡管理还款资金。核心需求：
- 账户余额管理（转入/转出，无限制）
- 交易流水记录（按日期倒序）
- 自动扣款：还款日当天自动从账户扣除月供
- 邮件提醒：提醒日与还款日一致（≤3天），邮件中加入账户余额

当前项目结构：
- 前端：React + TypeScript + Zustand + Tailwind CSS
- 后端：Cloudflare Workers API（loan-tracker-api.pages.dev）
- 自动任务：GitHub Actions（reminder/recalc.js）
- 导航：底部3个Tab（首页/计划/设置）

## 步骤 5：更新自动扣款逻辑

**文件：** `reminder/recalc.js`

在 `main()` 函数中，schedule 重算后、写回 API 前，新增自动扣款逻辑：

```javascript
function processAutoDeduction(data, schedule) {
  const account = data.repaymentAccount || { balance: 0, transactions: [] };
  const today = todayStr();

  for (const item of schedule) {
    if (item.date === today && !item.paid) {
      if (account.balance >= item.monthlyPayment) {
        account.balance = round2(account.balance - item.monthlyPayment);
        account.transactions.push({
          id: Date.now().toString(),
          date: today,
          type: 'repayment',
          amount: item.monthlyPayment,
          balanceAfter: account.balance,
          note: `第${item.period}期自动还款`
        });
        item.paid = true;
        console.log(`自动扣款：第${item.period}期 ¥${item.monthlyPayment}，余额 ¥${account.balance}`);
      } else {
        console.log(`余额不足：需要 ¥${item.monthlyPayment}，余额 ¥${account.balance}`);
      }
    }
  }

  data.repaymentAccount = account;
  return account;
}
```
